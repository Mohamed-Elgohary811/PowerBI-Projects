#### Overview
This Power BI dashboard provides insights into Spotify music data, including song popularity, artist performance, album type distribution, and monthly listening trends.

#### Key Metrics
- Distinct Songs
- Count of Artists
- Avg Duration (Minutes)
- Avg Popularity

#### Features
- Songs by artist breakdown
- Songs by popularity ranking
- Songs by album type (single vs album vs compilation)
- Explicit vs Non-Explicit songs distribution
- Distinct songs by year (2023 vs 2024)
- Avg popularity by album type
- Avg popularity trend by month
- Distinct songs count by month
- Interactive artist filter with album art

#### Tools Used
- Power BI
- Data Visualization
- Data Analysis
