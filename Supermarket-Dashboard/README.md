#### Overview
This Power BI dashboard provides insights into supermarket sales performance, customer behavior, and product analysis.

#### Key Metrics

- Total Sales
- No. of Items
- Average Rating
- Average Sales
- Item Visibility

#### Features

- Sales trends over time (by outlet establishment year)
- Fat content distribution analysis
- Item type performance insights
- Outlet size comparison
- Outlet location tier breakdown
- Fat content by outlet location type
- Outlet type performance table

#### Tools Used
- Power BI
- Data Visualization
- Data Analysis
